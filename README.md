# MP_springBootTest1

## boot工程依赖该maven插件打包

### 对应路径下执行target下的jar即可

#### java -jar demoBatis-0.0.1-SNAPSHOT.jar

```xml

<build>
        <plugins>
            <plugin>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-maven-plugin</artifactId>
            </plugin>
        </plugins>
</build>

```
